DrawingManager for p5.js

A drawing abstraction library for creative coding

DrawingManager is a p5.js library that provides a higher-level interface for creating and manipulating drawings. It records drawing operations as objects, allowing you to build complex drawings programmatically, apply transformations, and manage drawing state with ease.

Features

  • Record drawing operations as replayable objects
  • Build shapes incrementally with vertices
  • Apply transformations (translate, rotate, scale) to shapes
  • Programmatic manipulation of drawings
  • Clear and redraw capabilities
  • SVG export support (with p5.svg library)
